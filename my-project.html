My Seed Diversity Project

This page summarises what I did for my school project on seed diversity and what I learned from it.

Project goals

  • Explain seed diversity in a way that students can understand.
  • Show why it is important for climate, food and culture.
  • Create interactive tools like a store locator and seed comparison page.

What I created

Educational pages

I built different pages to explain concepts: what seed diversity is, why it matters, threats and actions.

Interactive tools

I added a seed tools page (gallery, filters, comparison, image analyzer) and a prototype store locator for seed access in Hyderabad.

Reflection

Through this project I learned both science and technology skills. I understood how important seeds are for the environment and people, and I practised building a complete website from scratch using HTML, CSS and JavaScript.
